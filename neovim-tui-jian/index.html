neovim介绍

**Neovim是一个旨在积极重构Vim源代码的项目，以实现以下目标：简化维护，提高bug修复和特性合并的速度。它诞生于2014年，是Vim editor的一个分支。**它的设计考虑到了现代用户，用Lua语言编写，目标是将代码模块化，使其更易于管理。

推荐原因：

Neovim近年来在开发人员和高级用户中获得了广泛的欢迎。它受欢迎的原因之一是可扩展性，它允许用户自定义和配置它以满足他们的需求。Neovim的插件使用“更通用的语言”——Lua，它是一种比Vimscript更通用的语言，允许您以更现代、更灵活的方式编写插件。neovim的主要优势：

可扩展性: Neovim是高度可扩展的，允许用户自定义和配置它，以满足他们的需求。在neovim上我们可以使用Packer插件管理器、gruvbox主题、LSP代码补全、文件和文本的模糊搜索、代码高亮等插件功能。

灵活性: Neovim是一个强大而灵活的文本编辑器，近年来在开发人员和高级用户中获得了很多人气。我们自定义各种快捷键，方便自己使用。

异步作业控制: Neovim具有异步作业控制，允许您在后台运行命令而不阻塞编辑器。该特性在LSP上体现明显。Neovim的LSP (Language Server Protocol)是一个插件，为内置的LSP客户端提供配置。它允许我们使用语言服务器为各种编程语言提供代码补全、语法检查和函数跳转等特性。

内置终端仿真器: ​Neovim有一个内置的终端仿真器，允许您直接在编辑器内运行命令。这可以避免我们来换切换编辑和命令窗口。
